SashaShop
A feature-rich Telegram e-commerce bot built with aiogram and SQLite, designed as a portfolio project to showcase modern asynchronous bot development.

Telegram E-Commerce Bot
A lightweight, fully functional Telegram shop bot created as a learning project and a portfolio piece to demonstrate asynchronous Python development and database integration.

Features
Product Catalog: Users can browse through various categories of items.
Visual Presentation: Every product includes high-quality images for customer evaluation.
Detailed Descriptions: Clear information and current pricing displayed for each item.
Seamless Ordering: Customers can place orders directly through the bot interface.
SQLite Database: Efficient data management for products, categories, and user orders.
Tech Stack
Language: Python 3.10+
Framework: aiogram (Asynchronous Telegram Bot API framework)
Database: SQLite (managed via aiosqlite / sqlite3)
Roadmap (Future Enhancements)
Integration of secure in-app payments (Telegram Payments API API / Stripe / LiqPay).
Admin panel for easy product and inventory management.
Order tracking system for customers.
