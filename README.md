# E-commerceBot

A feature-rich Telegram e-commerce bot built with aiogram and SQLite, designed as a portfolio project to showcase modern asynchronous bot development.

Telegram E-Commerce Bot
A lightweight, fully functional Telegram shop bot created as a learning project and a portfolio piece to demonstrate asynchronous Python development and database integration.

## Features
- Product Catalog:<br>
Users can browse through various categories of items.<br>
- Visual Presentation:<br>
Every product includes high-quality images for customer evaluation.
- Detailed Descriptions:<br> 
Clear information and current pricing displayed for each item.
- Seamless Ordering: <br>
Customers can place orders directly through the bot interface.
- SQLite Database:<br>
Efficient data management for products, categories, and user orders.

## Tech Stack
- Language: Python 3.10+
- Framework: aiogram (Asynchronous Telegram Bot API framework)
- Database: SQLite (managed via aiosqlite / sqlite3)
